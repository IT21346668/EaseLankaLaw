# EaseLankaLaw

**EaseLankaLaw** is an AI-driven legal chatbot for Sri Lanka that leverages **Large Language Models (LLMs)** to provide accurate legal information, support **Sinhala and English**, and improve access to justice.

---

## Project Objectives

### Main Objective

To develop an **AI-powered, multilingual legal assistance platform** that provides clear and simplified explanations of Sri Lankan laws.

### Sub Objectives

* Implement a legal information retrieval system using **Sri Lankan legal sources**
* Build a **lawyer recommendation system**
* Design a **user-friendly web application**
* Evaluate the system with a focus on **accuracy, usability, and legal awareness impact**
